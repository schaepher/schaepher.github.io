查看磁盘

fdisk -l

挂载

安装支持程序：

yum install -y ntfs-3g
mkdir /mnt/hd
mount -t ntfs /dev/sdb /mnt/hd

格式化磁盘

yum install -y ntfs-3g ntfsprogs
mkntfs -F --fast --label myDisk /dev/sdb
