InfluxDB 存储结构、读、写

发表于 2021-11-27 分类于 Database ， InfluxDB

介绍

InfluxDB 是开源的时序数据库。时序数据库的特点是数据按照时间顺序组织。

与 MySQL 的行式存储不同，InfluxDB 采用列式存储。在存储的时候，不同字段分开存储在不同的数据块中，一个字段的某个时间段的数据存放在磁盘中的连续位置。

InfluxDB 的代码库在 GitHub，地址是 https://github.com/influxdata/influxdb。截至 2021-11-17，其最新版本为 v2.1.1，但本文会以 v1.8.3 的代码为例子。

CPU

CPU 分两大类：Intel 和 AMD。

Intel 和 AMD 处理器都有一个面向中高端的主系列，Intel 是酷睿（Core，CPU 型号首字母为 i），AMD 是锐龙（Ryzen，CPU 型号首字母为 R）。Intel 的 i3、i5、i7、i9 和 AMD 的 R3、R5、R7、R9 对应。

在低端 CPU 市场中，Intel 是赛扬，AMD 是速龙。
