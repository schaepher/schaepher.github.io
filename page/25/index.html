Docker Compose

xxxx 发布于 2019-10-11 收录于 Learning Docker

Docker Compose 是用来做什么的

  1. 最直接的就是把原先要在命令行跑容器所需的参数整合到一个文件（docker-compose.yaml）里面，组织起来。这样就不用怕忘记某个参数了。
  2. 可以使用一行简单的命令（docker-compose）同时启动（up）、重启（restart）、关闭（stop）多个服务。

开发和测试环境使用另外安装的 docker-compose 命令，单机的生产环境也可用。
生产环境的集群如果是 Docker Swarm，则使用 Docker 自带的 docker stack deploy 。

raspberry-pi-4-raspbian

xxxx 发布于 2019-10-11

配置国内源

cp /etc/apt/sources.list /etc/apt/sources.list.default
sudo cat > /etc/apt/sources.list << DELIM
deb http://mirrors.tuna.tsinghua.edu.cn/raspbian/raspbian/ buster main non-free contrib 
deb-src http://mirrors.tuna.tsinghua.edu.cn/raspbian/raspbian/ buster main non-free contrib
DELIM

cp /etc/apt/sources.list.d/raspi.list /etc/apt/sources.list.d/raspi.list.default
sudo cat > /etc/apt/sources.list.d/raspi.list << DELIM
deb http://mirrors.tuna.tsinghua.edu.cn/raspberrypi/ buster main ui
DELIM

sudo apt-get update

CentOS7 用户管理

xxxx 发布于 2019-10-07 收录于 Installation and Configuration

基础操作

添加用户：
useradd -m username

-m 参数是为了给用户创建 home 目录。例如 /home/username

设置密码或修改密码：
passwd username

用户加组：
gpasswd -a username groupname

组移除用户： gpasswd -d username groupname

删除用户：
userdel username

CentOS7 NetworkManager 管理网络

工具

yum install -y NetworkManager-wifi

查看网卡状态

结果示例：

eth0           ethernet  connected     Wired connection 1 
wlan0          wifi      disconnected  --                 
p2p-dev-wlan0  wifi-p2p  disconnected  --                 
lo             loopback  unmanaged     --

如果是 unavailable，则重启系统。
