UUID

xxxx 发布于 2018-10-08 收录于 Development

https://stackoverflow.com/questions/20342058/which-uuid-version-to-use

  • version 3/5：
    根据某个命名空间和名称生成，名称相同则会产生相同的 UUID。可用于设置用户的 UUID（跟用户名唯一绑定）。一般用 version 5。

  • version 4:
    获取随机数。用得最多的一种。会重复的概率几乎可以忽略不计。
